  2. Übungsblatt 4
  • Schreiben Sie eine Funktion power, die über zwei Parameter verfügt, eine Fließkommazahl y und eine Ganzzahl x und deren Rückgabewert gleich yx ist. Beachten Sie, dass x positiv, negativ und 0 sein kann!

  • Schreiben Sie eine Funktion, die eine positive ganze Zahl (n) als Parameter hat und deren Rückgabewert n! ist. Sofern n negativ oder 0 ist, ist der Rückgabewert -1. Überprüfen Sie in der main-Funktion den Rückgabewert und erzeugen Sie eine Fehlermeldung, sofern notwendig.

  • Ändern Sie die Funktion von Aufgabe 4.2 so, dass das Ergebnis zusätzlich zum Rückgabewert in das beim Aufruf übergebene Argument geschrieben wird (Tipp: Referenzen verwenden).

  • Schreiben Sie eine Funktion maximum, die zwei Ganzzahlen als Parameter hat. Der Rückgabewert ist der größere der beiden Parameter (Verwenden Sie die bedingte Bewertung!).

  • Ändern Sie die Funktionen aus 4.4 und 4.5 so, dass jeweils Referenzen auf den größeren Parameter als Rückgabewert verwendet werden.

  • Schreiben Sie eine Funktion, mit einem Vektor von Ganzzahlen als Parameter, die die Elemente des (Original-) Vektors aufsteigend sortiert und das größte Element als Rückgabewert hat.

  • Schreiben Sie eine Funktion mit drei Parametern: einem vector vi und zwei Referenzen auf eine Fließkommazahl: mw und median. Die Funktion berechnet den Mittelwert und den Median von vi und gibt die Ergebnisse in den entsprechenden Parametern zurück. Sofern vi keine Elemente enthält ist der Rückgabewert -1, sonst 0. Überprüfen Sie in der main-Funktion den Rückgabewert und erzeugen Sie eine Fehlermeldung, sofern notwendig.
