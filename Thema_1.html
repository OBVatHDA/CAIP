Übungsblatt 0

  • Starten Sie ihre IDE. Erstellen Sie, wie in der Vorlesung gezeigt, ein leeres Projekt und fügen Sie eine leere C++-Datei hinzu.

  • Geben Sie das „Hello World“ - Programm aus der Vorlesung bzw. dem Skript ein.

  • Übersetzen Sie das Programm, beseitigen Sie mögliche Fehler und starten Sie das Programm.

  • Erzeugen Sie absichtlich Fehler im Quelltext, indem Sie z.B. ein Semikolon entfernen, übersetzen Sie das Programm und versuchen Sie die Fehlermeldungen zu verstehen.

  • Erstellen Sie ein neues Projekt und schreiben Sie ein Programm, das folgende Ausgabe erzeugt: Dies (“) ist ein Doppelhochkomma, und dies (\) ein backslash

  • Erstellen Sie ein neues Projekt und schreiben Sie ein Programm, das den Quellcode des „Hello World!“-Programms selbst als Ausgabe erzeugt.
