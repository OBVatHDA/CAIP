Übungsblatt 2

  • Schreiben Sie ein Programm, das die Zahlen von 10 abwärts bis -5 ausgibt, einmal mit einer while – Schleife und einmal mit einer for – Schleife.

  • Schreiben Sie ein Programm, das beliebig viele Zahlen einliest. Ist eine eingegebene Zahl durch 3 oder durch 5 teilbar, so soll sie wieder ausgegeben werden. Überlegen Sie sich, ob eine while- oder for- Schleife günstiger ist.

  • Erweitern Sie das Programm so, dass es am Ende zusätzlich die Summe der ausgegebenen Zahlen ausgibt.

  • Erweitern Sie das Programm so, dass es am Ende zusätzlich die größte eingegebene Zahl ausgibt.

  • Schreiben Sie ein Programm, das eine Anzahl von “*“ - Zeichen so ausgibt, dass ein Quadrat (z.B. 10 x 10), ein Rechteck (z.B. 20 x 10) und ein auf der Spitze stehendes Dreieck (10 Zeilen) entstehen. Die Figuren sollen jeweils mit “*“ - Zeichen gefüllt sein, die Anzahl der Zeilen / Spalten dürfen Sie selbst festlegen. Wenn Sie möchten erweitern Sie das Programm um die Eingabe der Seitenlänge(n).

  • Was macht das folgende Programm (zuerst überlegen, dann ausprobieren!):

    int i = 0;

    while (i++ < 10)

    {

      std::cout << i << std:endl;

    }

Optional:

  • Übersetzen Sie das Programm aus Kapitel 2.5 im Skript und starten Sie es

  • Ändern Sie das Programm so, dass es den Gruß ohne Zwischenraum zum Rahmen ausgibt.

  • Ändern Sie das Programm so, dass es den Anwender auffordert, die Anzahl der Zwischenräume einzugeben.

  • Das Programm schreibt die fast leeren Zeilen, die den Gruß von der ersten und letzten Zeile trennen, Zeichen für Zeichen. Ändern Sie das Programm so, dass es alle Leerzeichen mit einer einzigen Ausgabeanweisung erzeugt.
