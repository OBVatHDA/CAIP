  2. Übungsblatt 3
  • Nennen Sie drei Möglichkeiten einen vector mit 10 Elementen zu definieren, die jeweils alle den Wert 42 haben. Überprüfen Sie Ihre Vorschläge auf Richtigkeit. Welche Variante bevorzugen Sie und warum?

  • Schreiben Sie ein Programm, das die Zahlen von 1 bis 42 in einer Variablen des Typs vector abspeichert und geben sie zur Kontrolle alle Werte der Variablen aus (jeweils mit einer while- und mit einer for-Schleife).

  • Schreiben Sie ein Programm, das beliebig viele Zahlen einliest und anschließend den Median und den Mittelwert der eingelesenen Zahlen ausgibt. Bei einer geraden Anzahl von Zahlen ist der Median der Mittelwert der beiden mittleren Zahlen (nachdem die Zahlen sortiert wurden)

  • Ist der folgende Code lauffähig? Falls nicht verbessern Sie den Code.

      • vector vi;
      vi[0] = 42;

  • Wieviele Elemente sind in den folgenden vectoren, was ist ihr jeweiliger Wert? Beantworten Sie zuerst die Frage und überprüfen Sie Ihre Antwort, indem Sie ein Programm schreiben.

    1. vector v1;
    2. vector v2(10);
    3. vector v3(10, 42);
    4. vector v4{10};
    5. vector v5{10, 42};
    6. vector v6{10};
    7. vector v7{10, “42“};

  • Schreiben Sie ein Programm, das ein 'X' der Größe n ausgibt (wie in der Vorlesung gezeigt). Betrachten Sie das Kreuz als kleines digitales Bild, das aus n Zeilen und n Spalten besteht. Erzeugen Sie dieses Bild in dem Sie zeilenweise durch das Bild iterieren und an allen Pixeln ein Leerzeichen ausgeben, außer an den Stellen an denen sich das Kreuz befindet. Dort geben Sie ein beliebiges anders Zeichen aus.

  • Schreiben Sie ein Programm das beliebig viele Worte einliest. Die Eingabe wird beendet durch die Eingabe des Wortes Ende. Geben Sie die Worte in der Reihenfolge der Eingabe aus. Geben Sie danach die Worte alphabetisch sortiert aus.
